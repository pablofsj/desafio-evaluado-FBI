
# Desafío Evaluado FBI System

## 1. Instalación

Ejecutar el siguiente comando para instalar las dependencias del proyecto:
   ```bash
   npm i
   ```
   
## 2. Ejecución

Para ejecutar el proyecto de express utilizando nodemon, usar el siguiente comando:
   ```bash
   npm run dev
   ```

## 3. Info
